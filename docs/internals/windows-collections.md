# windows-collections — internals

> Maintenance notes for `windows-collections`. For usage, see the [guide](../guide/windows-collections.md).

- 📁 [Source](https://github.com/microsoft/windows-rs/tree/master/crates/libs/collections)

## How it's built

`src/bindings.rs` is generated by `tool_bindings` from `crates/tools/bindings/src/collections.txt`; the collection adapters are hand-written. Used internally by the `windows` crate.

# windows-result — internals

> Maintenance notes for `windows-result`. For usage, see the [guide](../guide/windows-result.md).

- 📁 [Source](https://github.com/microsoft/windows-rs/tree/master/crates/libs/result)

## How it's built

`src/bindings.rs` is generated by `tool_bindings` from `crates/tools/bindings/src/result.txt`. The `com`, `bstr`, and `strings` modules add COM error-info support and are gated on `windows` (and disabled under the `windows_slim_errors` cfg).

> Maintenance notes for `windows-core`. For usage, see the [guide](../guide/windows-core.md).

- 📁 [Source](https://github.com/microsoft/windows-rs/tree/master/crates/libs/core)

## How it's built

`src/bindings.rs` is generated by `tool_bindings` from `crates/tools/bindings/src/core.txt`. The hand-written modules (`agile_reference`, `com_object`, `compose`, `event`, …) provide the COM runtime support. The `implement`/`interface` proc macros are re-exported from `windows-implement`/`windows-interface`.

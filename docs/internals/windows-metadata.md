# windows-metadata — internals

> Maintenance notes for `windows-metadata`. For usage, see the [guide](../guide/windows-metadata.md).

- 📁 [Source](https://github.com/microsoft/windows-rs/tree/master/crates/libs/metadata)

## How it's built

Consumed by `windows-bindgen`. `src/bindings.rs` is generated by `tool_bindings` from `crates/tools/bindings/src/metadata.txt`; the ECMA-335 tables and readers are hand-written.

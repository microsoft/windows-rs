# windows-strings — internals

> Maintenance notes for `windows-strings`. For usage, see the [guide](../guide/windows-strings.md).

- 📁 [Source](https://github.com/microsoft/windows-rs/tree/master/crates/libs/strings)

## How it's built

`src/bindings.rs` is generated by `tool_bindings` from `crates/tools/bindings/src/strings.txt`; the string types and macros are hand-written.
